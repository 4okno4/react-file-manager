# React File Manager

![React](https://img.shields.io/badge/React-18-blue)
![TypeScript](https://img.shields.io/badge/TypeScript-5.x-blue)
![Vite](https://img.shields.io/badge/Vite-5.x-purple)
![Tailwind CSS](https://img.shields.io/badge/Tailwind_CSS-3.x-teal)

Современный учебный прототип веб‑интерфейса файлового менеджера с интуитивно понятным пользовательским интерфейсом. Проект разработан на **React 18** с использованием **TypeScript**, поддерживает drag-and-drop загрузку файлов, несколько режимов отображения, поиск, фильтрацию, сортировку и отображение статистики.

---

## Основные возможности

* **Два режима просмотра**: карточный и табличный
* **Drag & Drop загрузка**: перетаскивание файлов прямо в окно браузера
* **Поиск и фильтрация**: поиск по имени и фильтрация по типу файлов
* **Сортировка**: по имени, размеру и дате создания
* **Массовые операции**: выбор нескольких файлов и массовое удаление
* **Поддержка тем**: светлая и тёмная тема
* **Статистика**: сводная информация по текущему набору файлов
* **Предпросмотр**: быстрый просмотр выбранного файла (для локально загруженных изображений)
* **Адаптивный дизайн**: корректное отображение на мобильных устройствах

## Использование

### Добавление файлов

- Нажмите кнопку **«Добавить»** в панели управления
- Или перетащите файлы в окно браузера (Drag & Drop)

### Управление файлами

- **Поиск**: введите текст в поле поиска
- **Фильтрация**: выберите тип файла в выпадающем списке
- **Сортировка**: используйте переключатели сортировки
- **Предпросмотр**: кликните по имени или иконке файла
- **Удаление**: используйте кнопку удаления в карточке или таблице

### Массовые операции

- Отметьте чекбоксы рядом с файлами
- Используйте кнопку **«Удалить выбранные»**

---

## Архитектура и логика

- Управление состоянием реализовано с помощью **React Hooks**
- Производные данные (фильтрация, сортировка, статистика) вычисляются через `useMemo`
- Drag & Drop реализован на основе нативных событий браузера
- Архитектура подготовлена для дальнейшей интеграции с серверным REST API
